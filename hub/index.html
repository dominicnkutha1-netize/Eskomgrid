HOW THE REAL-TIME SYNC WORKS: All apps share data via a built-in engine (localStorage + BroadcastChannel). They auto-connect when open in the same browser. No internet, no server, no Firebase account needed for the demo.

FOR DIFFERENT PHONES: Host the folder on Netlify Drop (free, instant URL) — drag the folder to netlify.com/drop. All phones on same WiFi open the same URL — events sync across devices instantly.

PROJECTOR TIP: Put App04 (Eskom Ops) on the projector full-screen. Judges see the Eskom side as you trigger events. Maximum impact.
