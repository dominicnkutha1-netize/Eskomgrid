⚡ Power & Electrical Specifications

🔌 Main Power

Input: 220V AC
PSU Output: 5V DC @ 600mA
Module: HLK-PM01
Isolation: 3kV

🔋 Battery Backup

Cell: 18650 Li-Ion 3.7V
Capacity: 2600mAh
Backup: ~8–12 hours
Charger: TP4056

📡 Communication

Primary: WiFi 802.11 b/g/n
Backup: 2G GSM (SIM800L)
Protocol: HTTPS+WebSocket
Latency: <3 seconds

🌡️ Sensor Ranges

Voltage: 0–250V AC
Current: 0–30A AC/DC
Temperature: -55 to +125°C
RFID: 13.56MHz, 3–5cm

🔒 Security

Tamper: SW-420 vibration
Door: MC-38 reed switch
Auth: RFID Mifare 1K
Alarm: 95dB buzzer + LED

📏 Physical

Enclosure: 200×120×75mm
Rating: IP65 waterproof
Weight: ~350g assembled
Mount: Bolt-on/adhesive

😰 The Problem is BIGGER Than Load-Shedding

  • SA has 395,000+ distribution transformers — NONE automatically monitored
  • Copper theft costs Eskom R2+ BILLION per year
  • Transformer blows → people log calls → Eskom finds out HOURS later
  • Illegal electricity bridging costs R20+ BILLION annually
  • Vandalism leaves 10,000+ homes without power weekly
THE REALITY: Eskom has NO real-time visibility. NO automated alerts. NO tamper detection. Communities suffer in silence until enough calls pile up.

💡 Our Solution: GridGuard AI

🔒 Detects Unauthorized Access INSTANTLY

RFID authentication required. Vibration sensor detects forced entry. Magnetic door sensor detects unauthorized opening. Tamper = 🔊 SIREN + instant alert.

⚡ Monitors Transformer Health 24/7

Real-time voltage, current, temperature. Detects blown transformers in SECONDS. Identifies trip vs blown vs vandalism.

📱 Alerts Everyone IMMEDIATELY

Eskom control room gets instant dashboard alerts. Auto-generates fault tickets. Residents get area-specific push notifications.

🛡️ Prevents Electricity Theft

Detects illegal bridging through current anomalies. Auto-cutoff capability. Alerts investigation teams.

⚙️ How It Works — 4 Layers

LAYER 1: THE DEVICE (on transformer door) ESP32 + RFID + Voltage + Current + Temp + Door + Vibration Battery backup + GSM fallback → always reporting LAYER 2: THE CLOUD (real-time sync) Data syncs instantly → AI analyzes → Auto-generates tickets LAYER 3: ESKOM CONTROL ROOM Live dashboard → Instant alerts → Auto tickets → GPS dispatch Tamper/theft → SIREN + flashing alert LAYER 4: RESIDENT APP Area notifications → Power status → Ticket tracking
